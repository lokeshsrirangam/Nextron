Nextron is a multi-vendor e-commerce web application where multiple sellers can manage and sell their products on a single platform. The project focuses on building a clean shopping experience for customers, useful dashboards for vendors, and an admin panel to manage the entire system.

This project was built as a full-stack application to understand how real-world e-commerce platforms work.

---

## Features

- Multi-vendor system where vendors can add, update, and manage their own products  
- Vendor dashboards to track products, orders, and sales  
- Responsive customer storefront for browsing products and placing orders  
- Admin dashboard to manage vendors, products, and platform commissions  
- Secure authentication and role-based access for admin, vendors, and users  

---

## Tech Stack

- React.js and Next.js for the frontend  
- Tailwind CSS for styling and responsive design  
- Redux Toolkit for state management  
- Node.js and Express.js for backend APIs  
- MongoDB for database management
